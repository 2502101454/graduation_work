Django生成的数据表插入中文乱码
解决:
    原来自己的数据库在创建的时候没有指定编码方式，删除重建
    建新数据库的时候加上编码设置
		CREATE DATABASE mydb
 		DEFAULT CHARACTER SET utf8
  		DEFAULT COLLATE utf8_general_ci
  	然后重新生成数据表，现在就好了
